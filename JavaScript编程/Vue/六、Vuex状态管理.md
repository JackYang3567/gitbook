## 六、Vuex状态管理

Vuex的使用步骤：

### 安装 
 * 第一步，命令行安装:
  在终端（或命令行）运行安装，有如下两种方式：

#### NPM
```
npm install vuex --save
```
#### Yarn
```
yarn add vuex
```
 * 第二步：组织store文件结构：
 ```
 cd 项目/src
 mkdir store && cd store
 touch index.js
 ```

 * 编辑 src/store/indes.js文件，包含如下内容：

 ```
    import Vue from 'vue'
    import Vuex from 'vuex'

    Vue.use(Vuex)

 ```
